# 目标岗位模块

## 文件说明

- `JD001_公司名_岗位名.md`: 标准化后的岗位描述文件
- `JD002_公司名_岗位名.md`: 更多岗位描述...
- `templates/`: JD标准化模板（如果需要）

## 使用说明

1. 使用JD标准化脚本将网页上复制的岗位描述转换为统一格式
2. 脚本会自动分配JD ID（JD001, JD002...）
3. 生成的JD文件会保存在此文件夹
4. 即使不投递该岗位，也可以保存JD用于后续参考

## JD文件格式

每个JD文件包含：
- 基本信息（公司名、岗位名、工作地点、薪资等）
- 岗位描述
- 任职要求
- 其他相关信息
