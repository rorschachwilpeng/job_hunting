# 个人信息模块

## 文件说明

- `resume.pdf`: 原始简历PDF文件
- `resume_base.md`: 简历的Markdown文字版本（基础版本）
- `resume_versions/`: 针对不同岗位定制的简历版本
  - 命名格式：`resume_JD001_v1.md`（JD001表示岗位ID，v1表示版本号）
- `projects.md`: 详细的项目经历描述（用于生成简历和面试准备）

## 使用说明

1. 将PDF简历放在此文件夹
2. 将PDF内容转换为Markdown格式，保存为`resume_base.md`
3. 在`projects.md`中详细记录项目经历
4. 针对不同岗位生成的定制简历会保存在`resume_versions/`文件夹中
